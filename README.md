Revolutionizing Africa’s food systems. BioNovaa is an ecosystem bridging gaps in the blue economy and turning waste into wealth.

🚀 About The Project

This repository hosts the official BioNovaa Platform, a vibrant and professional web presence built to showcase our initiatives, partners, and impact.

Mission: To drive sustainable innovation by converting food system challenges into opportunities that uplift communities.

Key Features

🎨 Vibrant UI: Modern design with Tailwind CSS, featuring a "Navy & Green" palette and scroll animations.

📸 Interactive Gallery: A masonry-style gallery overlay that opens on demand.

☁️ Cloud Integration: Fully integrated with Firebase (Firestore & Storage) to allow real-time photo and video uploads from the community.

📱 Fully Responsive: Optimized for seamless experiences across mobile, tablet, and desktop.

🎥 Dynamic Content: Features video integration in the "About" section and gallery previews.

🛠️ Tech Stack

Frontend: HTML5, JavaScript (ES6+)

Styling: Tailwind CSS (CDN)

Icons: Lucide Icons

Backend / Database: Google Firebase (Firestore, Storage, Auth)

📂 Project Structure

index.html: Main application file containing all logic and styles.

BIONOVAA.jpg: Official Logo.

FTS.jpg: Partner Logo (Food Technologist Society).

BioNovaa_slideshow_video.mp4: Featured video asset.

🚀 How to Run Locally

Download the index.html file.

Place Assets: Ensure BIONOVAA.jpg, FTS.jpg, and BioNovaa_slideshow_video.mp4 are in the same folder as the HTML file.

Run: Double-click index.html to open it in your browser.
